# GlampTime - Система бронирования домов в лесу
## Описание проекта

**GlampTime** — это веб-приложение для бронирования домиков в глэмпинге. Отдых на природе с комфортом а так же различные активности на природе. Платформа позволяет пользователям просматривать каталог домиков\шатров, бронировать их, оставлять отзывы, заказывать дополнительные услуги. Администратор управляет домиками, бронированиями и модерацией отзывов через специальную панель управления. 
P.S - глэмпинг это разновидность кемпинга, объединяющая комфорт гостиничного номера с возможностью отдыха на природе. Гости размещаются в специально подготовленных объектах.
оборудованных удобствами:
- кровати;
- текстиль;
- wi-fi;
- мебель; 
- электричество; 
- сантехника.

### Основные возможности

**Для пользователя:**
- просмотр каталога домиков;
- фильтрация по цене и вместимости;
- бронирование с выбором дат;
- оставление отзывов с рейтингом;
- просмотр отзывов других гостей;
- регистрация и авторизация;
- настройка личного профиля;
- озакомление с услугами, а так же с различными активностями на природе.

**Для администратора:**
- управление домиками (добавление, редактирование, удаление);
- просмотр всех бронирований;
- изменение статуса бронирования;
- принятие и отклонение отзывов написаные пользователями;
- статистика по бронированиям;
- управление пользователями.

## Установка и запуск
### 1. Установите Node.js
Скачайте и установите Node.js с официального сайта: https://nodejs.org/

### 2. Установите PostgreSQL
Скачайте и установите PostgreSQL с официального сайта: https://www.postgresql.org/download/
При установке запомните пароль пользователя postgres

### ВАЖНО- перед скачиванием node.js и postgre.sql, выбирите последниее версии

### 3. Скачайте проект
Для того чтобы скачать проект, клонируем репозиторий и переходим проект черех терминал vs code\ cmd\windowsPowerShell
git clone https://github.com/ваш-аккаунт/glamptime.git
cd glamptime

### 4. Установите зависимости
Устанавливаем зависимости с помощью команды:
- npm install

### 5. Создайте базу данных
Откройте командную строку и выполните следущую команду:

psql -U postgres

Введите пароль, затем создайте базу:
sql
CREATE DATABASE glamptime_db;
Далее требуется перейти в базу данных glamptime_db с помощью команды \q  

### 6. Настройте подключение к БД
Откройте файл server.js и в строке с подключением укажите ваш пароль:

javascript
const pool = new Pool({
    host: 'localhost',
    port: 5432,
    user: 'postgres',
    password: 'СЮДА ВВЕДИТЕ СВОЙ ПАРОЛЬ',  
    database: 'glamptime_db',
});

### 7. Создание нового терминала и переход к папке проекта
cd папка в которой вы находитесь\ указываем путь к проекту.
Пример: PS D:\птк сесии\курсач3 курс> cd glampTime  


### 8. Запуск сервера
После перехода в папке с проектом, вводим в терминал следущую команду, для запуска сервера
- node server.js

### 9. Откройте сайт
Сервер успешно запущен. 
Перейдите в браузере по адресу: http://localhost:3000. Нажав сочетание клавиш Crt + правая кнопка мыши
