# GlampTime - Система бронирования домов в лесу
## Описание проекта

**GlampTime** — это веб-приложение для бронирования домиков в глэмпинге (отдых на природе с комфортом). Платформа позволяет пользователям просматривать каталог домиков\шатров, бронировать их, оставлять отзывы, заказывать дополнительные услуги. Администратор управляет домиками, бронированиями и модерацией отзывов через специальную панель управления. 
P.S - глэмпинг это разновидность кемпинга, объединяющая комфорт гостиничного номера с возможностью отдыха на природе. Гости размещаются в специально подготовленных объектах, 
оборудованных удобствами:
- кровати,
- текстиль,
- мебель, 
- электричество, 
- сантехника, 
- иногда услуги питания. 

### Основные возможности

**Для пользователя:**
- Просмотр каталога домиков
- Фильтрация по цене и вместимости
- Бронирование с выбором дат
- Оставление отзывов с рейтингом
- Просмотр услуг и их заказ
- Просмотр отзывов других гостей
- Регистрация и авторизация
- Настройка личного профиля
- Озакомление с услугами, а так же с различными активностями на природе

**Для администратора:**
- Управление домиками (добавление, редактирование, удаление)
- Просмотр всех бронирований
- Изменение статуса бронирования
- Модерация отзывов 
- Статистика по бронированиям
- Управление пользователями

- **Backend:** Node.js, Express.js

- **API:** REST API
- **Язык программирования:** JavaScript (ES6+)

## Установка и запуск
### 1. Установите Node.js
Скачайте и установите Node.js с официального сайта: https://nodejs.org/

### 2. Установите PostgreSQL
Скачайте и установите PostgreSQL с официального сайта: https://www.postgresql.org/download/
При установке запомните пароль пользователя postgres

### 3. Скачайте проект

git clone https://github.com/ваш-аккаунт/glamptime.git
cd glamptime

### 4. Установите зависимости
npm install

### 5. Создайте базу данных
Откройте командную строку и выполните следущую команду:

psql -U postgres

Введите пароль, затем создайте базу:
sql
CREATE DATABASE glamptime_db;
\q

### 6. Настройте подключение к БД
Откройте файл server.js и в строке с подключением укажите ваш пароль:

javascript
const pool = new Pool({
    host: 'localhost',
    port: 5432,
    user: 'postgres',
    password: 'ВАШ_ПАРОЛЬ',  
    database: 'glamptime_db',
});

### 7. Запуск сервера

 cd папка в которой вы находитесь/ указываем путь к проекту
node server.js

### 8. Откройте сайт
Сервер успешно запущен. Перейдите в браузере по адресу: http://localhost:3000
